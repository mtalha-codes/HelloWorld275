# C++ Practice Tests Overview

This repository contains practice problems grouped by chapters. Each chapter has five problems:
- Problems 1–3: Easy.
- Problem 4: Intermediate.
- Problem 5: Mini-project where necessary.

How to use:
1. Pick a chapter file below.
2. Solve problems in order. Try to write small programs that read from stdin and write to stdout.
3. Keep solutions simple and use only concepts of the chapter.

Click to test yourself chapter wise:
> [!WARNING]
> These tests expects you to be familiar with the notes given in the release section of the repository.
> I would advise you to first go through the notes and then problems because these problems uses all the concepts 
> from the notes and are designed to test your understanding of those concepts.

- [Chapter 1–5](problems_ch1_ch5.md) 
- [Chapter 6–8 ](problems_ch6_ch8.md)
- [Chapter 9–10](problems_ch9_ch10.md)

Difficulty rules are strict:
- the first problems guide practice of basics with a small challenge,
- the 4th stretches your skill,
- and the 5th is an integrative task (mini-project) where applicable.

Good luck and solve one problem at a time.
